# 🏛️ ЭтноУрок — Конструктор образовательных сценариев

**Автор:** Pivnoy Anchous

Платформа для студентов педагогических вузов и учителей, которая помогает мгновенно создавать увлекательные мероприятия о культуре и традициях народов России с помощью искусственного интеллекта.

---

## ✨ Основные возможности

- **Интеллектуальная генерация:** Создание уникального сценария за 15-30 секунд на основе темы и параметров.
- **Поминутный тайминг:** Каждый этап урока (вступление, основная часть, рефлексия) расписан по минутам в зависимости от общей длительности.
- **Учет контекста:** Программа адаптирует сложность под возраст (начальная, средняя, старшая школа) и доступные инструменты (интерактивные доски, смартфоны и т.д.).
- **История запросов:** Ваши последние 10 генераций сохраняются локально в браузере.
- **Печатная форма:** Оптимизированные стили для печати готового плана урока прямо из браузера.

## 🛠 Технологический стек

- **Frontend:** HTML5, CSS3 (Modern UI, Google Fonts, FontAwesome), JavaScript (ES6+).
- **Backend:** Node.js, Express.
- **AI Integration:** OpenRouter API (модель DeepSeek / Gemini).
- **Библиотеки:** `marked.js` для рендеринга методических материалов.

## 📋 Структура генерируемого сценария

Программа выдает результат по строгому стандарту (7 разделов):
1. Название урока.
2. Целевая возрастная группа.
3. Цели и задачи.
4. Вступление (с таймингом).
5. Основная часть (поминутный план активностей).
6. Рефлексия.
7. Главный вывод.

## 🚀 Как запустить проект локально

### 1. Подготовка бэкенда
Перейдите в папку с сервером и установите зависимости:
```
npm install express cors node-fetch
```
### 2. Настройка API
Откройте файл ai.js и вставьте ваш ключ доступа в переменную API_KEY:
```
const API_KEY = "KEY-HERE";
```
### 3. Запуск
Запустите сервер:
```
node server.js
```
